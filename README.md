# Trabalhos de AI

Uma coleção de trabalhos realizados no meu 12º ano para a disciplina de Aplicações Informáticas desenvolvidos em p5.js.

Podem existir problemas de compatibilidade com novas versões do p5, então pode ser preciso mudar para uma versão mais antiga para testar.

- **Lançamento oblíquo:** programa para calcular os valores acossiados ao lançamento oblíquo de um objeto (neste caso, o Faísca McQueen).

- **Xadrez:** um conjunto de mini-jogos (diferenças, labirinto, xadrez) criados durante as aulas, e especificamente para o trabalho. Também incluí um video gravado para este trabalho. 

- **Fusion Invasion:** Um raycaster desenvolvido como projeto final para a disciplina. Incluí 3 níveis, 3 armas diferentes e um menu de definições (e de cheats).
